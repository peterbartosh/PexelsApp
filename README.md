# PexelsApp
Photo catalog app.

# Stack: 
- Local Storage: Room.
- DI: Hilt.
- Networking: Retrofit, OkHttp.
- Image Loading: Coil.
- Concurrency: Coroutines + Flow.
- Navigation: Jetpack Navigation.
- Ui: Jetpack Compose.
